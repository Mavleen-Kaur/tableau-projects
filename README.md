# Tableau Projects Portfolio

This repository contains three Tableau dashboard projects focused on practical data analysis.  
Each project folder includes the Tableau workbook, dataset, and dashboard screenshots.

---

# 1. Road Accident Analysis

**Objective:** 
Analyze road accident data to identify major causes, accident-prone locations, and risk factors.

**Key Insights & KPIs:**  
- Total accidents by region  
- Accidents by weather, lighting, and road conditions  
- Vehicle type involvement  
- Severity analysis (fatal, serious, minor)  
- Time-of-day and day-of-week patterns  

## 2. Employee Analysis 

**Objective:**  
Evaluate workforce trends and identify factors such as attrition, performance, satisfaction, and workforce distribution.

**Key Insights & KPIs:**  
- Employee attrition rate  
- Age group and department distribution  
- Salary vs job satisfaction  
- Performance metrics  
- Work-life balance indicators  

##  3. Environmental Impact Analysis

**Objective:**  
Examine environmental indicators such as pollution levels, emissions, or resource consumption to assess environmental trends.

**Key Insights & KPIs:**  
- Air quality metrics (PM2.5, PM10, NO2, etc.)  
- Pollution by region/time  
- Emission trends    
- Year-over-year comparison
